# AmazonAlexAnalysis
 -This dataset consists of a nearly 3000 Amazon customer reviews (input text), star ratings, date of review, variant and feedback of various amazon Alexa
 products like Alexa Echo, Echo dots, Alexa Firesticks etc.
 
 -A simple data analysis has been made to discover insights into consumer reviews.
 
 -Model has been trained for sentiment analysis and analyze customer reviews to see how many positive reviews ? and how many negative reviews ?
